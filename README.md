# PagCall - Módulo Call Center

Protótipo visual do módulo de Call Center da plataforma **pagah**.

## Páginas

| Página | Descrição |
|--------|-----------|
| [index.html](index.html) | Hub de navegação do módulo |
| [leads-atendimento.html](pages/leads-atendimento.html) | Gestão de leads e atendimentos |
| [gestao-empresas.html](pages/gestao-empresas.html) | Cadastro e gestão de empresas |
| [gestao-usuarios.html](pages/gestao-usuarios.html) | Gestão de usuários e equipes |
| [dashboard-vendedor.html](pages/dashboard-vendedor.html) | Dashboard do vendedor |
| [dashboard-gerente.html](pages/dashboard-gerente.html) | Dashboard do gerente |

## Stack

- HTML estático (protótipo visual)
- Tailwind CSS via CDN
- Google Material Symbols Rounded
- Fonte Ubuntu (Google Fonts)

## Como usar

Abra `index.html` no navegador ou acesse via Vercel.
